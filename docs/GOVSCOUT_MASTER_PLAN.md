GovScout.pro is an AI-powered federal contracting platform.

Core goal:
Help small businesses find SAM.gov opportunities, understand contracts, prepare bid materials, and receive alerts.

Main features:
1. User onboarding
2. Business profile
3. SAM.gov contract search
4. Saved searches
5. Email and Telegram alerts
6. Marcus AI contract analysis
7. Proposal package generator
8. AppSumo-ready subscription access

Primary stack:
Frontend: Next.js + Tailwind + TypeScript
Hosting: Netlify
Database/Auth: Supabase
AI: OpenAI/Claude API
Repo: GitHub

Rule:
