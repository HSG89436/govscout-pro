# GovScout.pro

AI-powered federal contracting platform for small businesses.

## Project Foundation

This repository is the central workspace for GovScout.pro engineering, QA, marketing, roadmap, deployment, and launch readiness.

## Folder Structure

- `docs/` - product requirements, roadmap, launch checklist, QA standards, and engineering standards
- `prompts/` - reusable AI agent prompts and operating instructions
- `reports/` - QA, marketing, engineering, and launch readiness reports
- `tasks/` - implementation tasks, sprint notes, and work tracking
- `src/` - website and application source files
- `netlify/functions/` - Netlify serverless functions

## Netlify Deployment

Netlify is configured with `netlify.toml`.

- Publish directory: `src`
- Functions directory: `netlify/functions`
- Build command: none required for the current static starter
